# Pasta de Vídeos

## Instruções

Para adicionar o vídeo do hero:

1. Baixe um vídeo de café de alta qualidade de:
   - Pexels: https://www.pexels.com/pt-br/videos/cafe/
   - Pixabay: https://pixabay.com/pt/videos/
   - Unsplash: https://unsplash.com/pt-br/videos

2. Renomeie o vídeo para: `coffee-hero.mp4`

3. Coloque o arquivo nesta pasta

**Vídeo recomendado:**
- Resolução: 1920x1080 (Full HD)
- Duração: 10-30 segundos
- Formato: MP4 (H.264)
- Tamanho: < 5MB para melhor performance

**Alternativa:**
Você também pode usar URLs de CDN como Pexels ou usar o fallback para imagem de fundo que já está configurado no CSS.
